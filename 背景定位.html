让元素的背景图定位在右下角距离左边 20px ,距离下面 10px

Rocket

主要代码

利用 css3 对 background-position 的扩展功能,加之前的关键字之后添加我们的具体偏移量来实现这一功能

 
.ways-one {
	/* css 3 */
	background: url(img/rocket.svg) no-repeat yellowgreen;
	background-position: right 20px bottom 10px;

	/* 回退方案 */
	background: url(img/rocket.svg) no-repeat right bottom yellowgreen;
	background-position: right 20px bottom 10px;
}
Rocket

主要代码

利用 css3 对 background-origin 的功能,让背景的定位根据我们的预期来定位.

  • border-box 以边框定位背景
  • padding-box(默认)以内边距定位背景
  • content-box 以内容定位背景
 
.ways-two {
	padding: 0 20px 10px 0;
	background: url(img/rocket.svg) no-repeat right bottom yellowgreen;
	background-origin: content-box;
	box-sizing: border-box;
}

主要代码

使用 calc 计算方法

 
.ways-three {
	background: url(img/rocket.svg) no-repeat yellowgreen;
	background-position: calc(100% - 20px) calc(100% - 10px);
}
